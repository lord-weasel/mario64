How to Play

This is a web-based version of Super Mario 64 compiled with Emscripten. The game runs entirely in your browser using WebAssembly and WebGL.

Controls: Use your keyboard and mouse to control Mario. Arrow keys for movement, spacebar for jumping.

Note: This is a demonstration build. Full game functionality requires additional development and integration with the complete SM64 codebase.
